## Prérequis

- [Node.js](https://nodejs.org) v18 ou supérieur
- npm

## Installation

```bash
cd svg-editor

npm install

npm run start
```

## Créer un exécutable

Génération avec electron-forge : 

```bash
npm run make
```

L'exécutable sera généré dans le dossier `out/`.

Pour Windows : un installateur `.exe` sera créé.  
Pour macOS : un `.dmg` sera créé.  
Pour Linux : un `.AppImage` sera créé.
